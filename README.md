 # 原码 反码 补码
 - 在计算机中通常使用补码，其操作就是在反码的基础上+1，这样的话就不存在＋－0的情况

 # 大端和小端
 - 大端一般是在arm x86 上面比较常见，就是从小大到依次存储，小端则相反，主要就是自己打印地址的时候知道顺序

 # 数值类型
 - 负数如果直接使用unsigned则会变成一个很大的数，这是因为补码存储形式和字节解释导致的
